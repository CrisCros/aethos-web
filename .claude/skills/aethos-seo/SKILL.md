# Aethos SEO Skill

Optimize for:
- Spain
- Romania
- future Germany
- local businesses
- service businesses

Aethos routes:
- /
- /services
- /work
- /about
- /contact

Future SEO pages:
- /web-design
- /seo
- /landing-pages
- /automation
- /industries/clinics
- /industries/restaurants
- /industries/real-estate
- /industries/construction

Every page must include:
- one clear H1
- SEO title under 60 characters
- meta description under 160 characters
- clean heading hierarchy
- semantic HTML
- internal links
- descriptive links
- service-focused content
- mobile-first layout
- readable copy for business owners

Avoid:
- keyword stuffing
- duplicated titles
- empty metadata
- vague SaaS copy
- fake testimonials
- fake project results

Write for humans first, Google second.
