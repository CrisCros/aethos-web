# Client Website Skill

Every client website should focus on:
- clear service explanation
- trust
- fast contact
- mobile-first design
- SEO basics
- local search
- simple lead capture
- WhatsApp/contact visibility

Required sections:
- hero with clear offer
- services
- why choose us
- location/service area
- contact CTA
- contact form
- WhatsApp/contact button
- footer with business info

For local SEO:
- include city/area naturally
- include service keywords
- include Google Business Profile guidance
- add metadata
- add FAQ if useful

Avoid:
- generic templates
- fake reviews
- fake photos
- too much animation
- hidden contact info
- vague service descriptions

Goal:
Build websites that help real local businesses get contacted.
