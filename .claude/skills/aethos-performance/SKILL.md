# Performance Review

Check:
- image sizes
- unused JavaScript
- font loading
- CSS bloat
- hydration issues
- layout shift
- render-blocking resources
- large animations
- unnecessary libraries
- Lighthouse performance
- mobile performance

Rules:
- optimize images
- use next/image where appropriate
- avoid huge client components
- prefer CSS animations for simple effects
- only animate transform and opacity
- lazy-load heavy sections
- avoid unnecessary Framer Motion usage
- keep pages fast on mobile

Goal:
The site should feel fast, not just look good.
