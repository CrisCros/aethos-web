# Aethos Dark Mode Audit

Check every page and component for:
- text contrast
- card contrast
- background contrast
- button states
- muted text visibility
- borders
- badges
- form fields
- icons
- footer readability
- project cards
- service cards
- hero sections

Never use:
- white text on pale cards
- low-opacity gray on dark backgrounds
- muted labels that disappear
- light success badges on dark cards
- decorative gradients that reduce readability

Dark mode should feel:
- premium
- readable
- consistent
- warm
- intentional

Every section must be readable in both light and dark mode.

After changes:
- verify contrast manually
- check mobile dark mode
- check navbar and footer
- check project cards
- check services page
