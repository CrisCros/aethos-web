# Accessibility Review

Check:
- color contrast
- keyboard navigation
- visible focus states
- alt text
- heading order
- button labels
- form labels
- error states
- link text clarity
- mobile tap targets
- reduced motion
- readable font sizes

Avoid:
- low contrast text
- icon-only buttons without labels
- divs acting as buttons
- missing form labels
- heading jumps
- tiny text
- hover-only interactions
- animations without prefers-reduced-motion

Every site should be usable by keyboard and screen readers.
