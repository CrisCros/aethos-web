# Human UI Review

Use this skill when the design feels:
- too flat
- too white
- too corporate
- too SaaS
- too AI-generated
- too repetitive
- too templated

Check for:
- generic gradients
- repeated card patterns
- excessive bento grids
- weak contrast
- cold copy
- too many dashboards
- fake-looking visuals
- lack of brand personality
- too much empty white space without rhythm

Improve with:
- stronger color personality
- warmer backgrounds
- editorial layouts
- asymmetry
- human copy
- real business situations
- visual rhythm
- section contrast
- fewer but better animations
- more memorable visual details

The website should feel:
- human
- warm
- modern
- trustworthy
- practical
- lively
- not corporate
